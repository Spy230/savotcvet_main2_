# Сайт Автосервиса

Современный информационный сайт автосервиса, разработанный на основе технического задания версии 1.2.

## Технологии

- **React 18** с TypeScript
- **Vite** для быстрой сборки
- **TailwindCSS** для стилизации
- **Lucide React** для иконок
- Полностью адаптивный дизайн (mobile-first)

## Структура сайта

1. **Главная** - Hero секция с ключевыми преимуществами
2. **Услуги** - Полный прайс-лист с категориями услуг
3. **Наши работы** - Галерея выполненных работ
4. **Сертификаты** - Документы и лицензии
5. **О компании** - Информация о компании и статистика
6. **Контакты** - Контактная информация и карта

## Особенности

- Быстрая загрузка (оптимизация под PageSpeed)
- Lazy loading для изображений
- Семантическая HTML разметка
- SEO-оптимизация (мета-теги, Open Graph, Twitter Cards)
- Кроссбраузерная совместимость
- Доступность (a11y) - базовый уровень
- Плавная навигация по якорям
- Мобильное меню
- Кликабельные контакты (tel:, mailto:)

## Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Превью продакшен сборки
npm run preview
```

## Структура папок

```
src/
├── components/        # Компоненты страницы
│   ├── Header.tsx    # Шапка сайта с навигацией
│   ├── Hero.tsx      # Главный экран
│   ├── Services.tsx  # Раздел услуг и цен
│   ├── Gallery.tsx   # Галерея работ
│   ├── Certificates.tsx # Сертификаты
│   ├── About.tsx     # О компании
│   ├── Contacts.tsx  # Контакты
│   └── Footer.tsx    # Подвал сайта
├── App.tsx           # Главный компонент
└── main.tsx          # Точка входа

public/
├── images/           # Папка для изображений
│   ├── gallery/      # Фото работ
│   ├── certificates/ # Сертификаты
│   └── placeholder/  # Заглушки
├── robots.txt        # Для поисковых роботов
└── sitemap.xml       # Карта сайта
```

## Оптимизация

- Все изображения должны быть в формате WebP
- Размер изображений: до 300-500 КБ
- Используйте атрибут loading="lazy" (уже реализовано)
